# GradeSync - CGPA Calculator & What-If Simulator

Welcome to GradeSync, a modern, premium web application built for students to calculate their Semester Grade Point Average (SGPA) and Cumulative Grade Point Average (CGPA). 

## Features

- **Semester Management:** Easily add and remove semesters.
- **Subject Tracking:** Input subject names, credits, and grades (using a standard 10-point scale: O, A+, A, B+, B, C, P, F).
- **Real-Time Calculation:** SGPA and overall CGPA update instantly as you enter your data.
- **What-If Mode:** Curious about your final CGPA if you score an 'A' in all subjects next semester? Toggle the "What-If" switch to add simulated semesters. Simulated semesters project your future CGPA without affecting your actual record.
- **Premium Aesthetics:** Built with a beautiful, responsive, glassmorphism-inspired dark mode UI.

## How to Run

GradeSync is a client-side Single Page Application (SPA). No backend or build step is required!

1. Open the folder containing the project.
2. Double-click on `index.html` to open it in your default web browser.
3. Start calculating your grades!

## Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Styling using CSS Variables, Flexbox, Grid, and modern design principles (glassmorphism).
- **Vanilla JavaScript:** DOM manipulation and pure client-side state management.
